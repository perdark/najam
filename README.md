# نجمة وكمر — موقع التوظيف

موقع صفحة واحدة (Landing) لاستقطاب فريق المناسبات، مع استمارة توظيف متعددة الخطوات
ترسل كل طلب مباشرة إلى **بوت تلكرام**.

- **الواجهة:** HTML + CSS + JavaScript صِرف (بدون أي framework) — سريع جداً على الهواتف الضعيفة.
- **الخادم:** دالة واحدة فقط `api/submit.js` (Serverless على Vercel) تحفظ توكن البوت بالخادم ولا تكشفه للمتصفح.
- **5 وظائف:** مصورة هاتف · منسقة زهور · كول سنتر وإدارة الحجوزات (للبنات) — تصميم جرافيك · موارد بشرية (للجميع: شباب وبنات). أقسام الأسئلة تظهر حسب الوظيفة المختارة، ويُطلب الجنس في المعلومات الأساسية.

## البنية
```
index.html        الصفحة + الاستمارة (الخطوات مكتوبة كاملة)
styles.css        التصميم (ثيم فاتح: موف + ذهبي، مطابق للوغو)
script.js         الأنميشن + منطق الخطوات + الإرسال
api/submit.js     استقبال الطلب وإرساله لتلكرام (التوكن بالـ env)
assets/           اللوغو والأيقونات (مستخرجة من صورة اللوغو)
.env.example      نموذج المتغيرات السرية
```

## الإعداد (مرة وحدة)

### 1) أنشئي بوت تلكرام
1. افتحي **@BotFather** على تلكرام → `/newbot` → خذي **TOKEN**.
2. أنشئي مجموعة (أو محادثة) لاستلام الطلبات، وأضيفي البوت لها.
3. أرسلي أي رسالة، ثم افتحي بالمتصفح:
   `https://api.telegram.org/bot<TOKEN>/getUpdates`
   وخذي قيمة `chat.id` (للمجموعات تكون رقم سالب).

### 2) المتغيرات السرية
انسخي `.env.example` إلى `.env.local` وعبّي:
```
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
```

### 3) روابط التواصل (اختياري)
في أعلى `script.js` بـ `CONFIG` غيّري:
- `INSTAGRAM_URL` → رابط بيج الانستغرام.
- `WHATSAPP_NUMBER` → رقم الواتساب بالصيغة الدولية (بدون +).

## التشغيل محلياً
يحتاج Vercel CLI لأن الاستمارة تستعمل دالة سيرفر:
```bash
npm i -g vercel
vercel dev
```
ثم افتحي `http://localhost:3000`.

> معاينة الشكل فقط (بدون إرسال) تكفيها أي خادم ثابت، مثل: `python3 -m http.server`.

## النشر على Vercel
```bash
vercel            # معاينة
vercel --prod     # نشر مباشر
```
ثم في **Vercel → Project → Settings → Environment Variables** أضيفي
`TELEGRAM_BOT_TOKEN` و `TELEGRAM_CHAT_ID` (للـ Production)، وأعيدي النشر.

## ملاحظات أمان
- التوكن يبقى بالخادم فقط (env)، ولا يصل المتصفح أبداً.
- يوجد حقل فخّ (honeypot) ضد البوتات، وتحقق من صحة البيانات في الخادم.
- لا حاجة لتسجيل دخول جوجل ولا رفع ملفات — المتقدمة ترسل أعمالها على واتساب بعد التقديم.
